# C++ / R / Python Cheatsheet
## Conditions
#### C++
```
if (condition) {
  // Statements
}
else if (condition) {
  // Statements
}
else {
  // Statements
}
```
#### R
```
if (condition) {
  // Statements
} else if (condition) {
  // Statements
} else {
  // Statements
}
```
#### Python
if condition:
  // Statements
elif condition:
  // Statements
else:
  // Statements
## Loops
### For
#### C++
```
for (statement1; condition; statement2) {
  // Statements
}
```
```statement1``` is executed only the first time the loop is executed. ```statement2``` is executed every time at the end of the loop.
#### R
```
for (value in sequence) {
  // Statements
}
```
#### Python
```
for x in range(0,3):
  // Statements
```
### While
#### C++
```
while (condition) {
  // Statements
}
```
#### R
```
while (condition) {
  // Statements
}
```
#### Python
```
while condition:
  // Statements
```

### Do-while
#### C++
```
do {
  // Statements
} while (condition);
```
#### R
```
repeat {
  // Statements
  if (condition) { break }
}
```
#### Python
```
while True:
  // Statements
  if condition: break
```
